# ChatMe

ChatMe is a real-time messaging application that enables users to communicate instantly through a modern and responsive chat interface. The platform provides secure authentication, real-time message delivery, and a seamless user experience across devices.

## Live Demo

[Start Chatting!!!](https://chat-me-rtjj.onrender.com)

## Features

* User authentication and authorization
* Real-time messaging
* One-to-one conversations
* Online and offline user status
* Responsive user interface
* Secure session management
* Modern chat experience

## Tech Stack

### Frontend

* React.js
* Tailwind CSS

### Backend

* Node.js
* Express.js
* Socket.IO

### Database

* MongoDB

## How It Works

1. Users create an account or log in.
2. Users can view available conversations.
3. Messages are delivered instantly using WebSockets.
4. Online status updates in real time.
5. Conversations are stored and accessible whenever users return.

## Project Highlights

* Real-time bidirectional communication
* Responsive design for desktop and mobile devices
* Efficient message handling using WebSockets
* Secure authentication workflow
